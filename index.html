Sobre o sistema

Fluxvid é um sistema de fluxo de dados contínuo usando o padrão DASH. Sistema esse que foi desenvolvido para a cadeira de graduação Redes Multimidias. Foi gravado um vídeo para realização dos testes do sistema.

Ele pode ser acessado através da busca no topo.
